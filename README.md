* ### Task name: HTTP(S) Webserver
* ### This is a http(s) server that accepts the url to the site in any encoding and saves all images from it.

* ### Run it
    #### Linux:
        python3 server.py -path [--port] [-help]
    #### Windows:
        python server.py -path [--port] [-help]
